Níveis de Ensino

Básico (A1-A2)

Ideal para iniciantes que desejam aprender o inglês do zero ou têm conhecimentos básicos.

  • Vocabulário essencial
  • Gramática básica
  • Conversação simples
  • Compreensão de textos básicos

Duração estimada: 6-8 meses

Intermediário (B1-B2)

Para alunos que já têm uma base sólida e desejam melhorar a fluência.

  • Vocabulário avançado
  • Gramática intermediária
  • Conversação fluente
  • Compreensão de textos complexos
  • Redação de textos

Duração estimada: 8-10 meses

Avançado (C1-C2)

Para alunos que buscam proficiência completa e preparação para exames internacionais.

  • Vocabulário especializado
  • Gramática avançada
  • Conversação nativa
  • Compreensão de textos acadêmicos
  • Redação acadêmica
  • Preparação para TOEFL/IELTS

Duração estimada: 10-12 meses
